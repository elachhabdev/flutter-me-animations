# Flutter ME Animations

Great App for Learning Animation in flutter

Flutter ME Animations is a Journey That make you an expert in flutter animation.

The purpose of this App is to teach you how Animation are done in flutter ,There is +50 animations and still others coming

## why this App is a great for taking your skills of UI/UX in to the next level.

---

###  :racehorse: Animations 

are chosen so that teach you something new and deep about the techniques behind it

the animations either increase in complexity or introduce new technique and are so versatile

after you finish and we recommande you to invest your time to rebuild them because it let you build intuition how to handle any new one 


###  :slot_machine: Patterns 

the App will teach you the concept of a lot of complex UI/UX 
and let you figurate new one by yourself
  
  - Gesture driven
  - Scroll driven
  - Physics driven
  - Ticker driven
  - Hero driven
  - State driven
  - And More


###  :bird: Flutterly

the App use the power of flutter to solve the challenges presented by the animation for instance:

   - we use AnimatedWidget
   - we use ImplicitlyAnimatedWidget
   - we use AnimationController
   - we use Physics simulation:FrictionSimulation,SpringSimulation
   - we use Ticker for low level
   - we use Gesture handlers
   - we use Transform
   - we use RenderObject
   - we use RenderSliver
   - we use Shaders
